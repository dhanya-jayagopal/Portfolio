# Dhanya Jayagopal — Portfolio

Personal portfolio and case study site for Dhanya Jayagopal, Founding PM at Bomisco. Built to document product work across enterprise SaaS, platform, and growth — and to write about it.

**Live:** [portfolio-dhanya-jayagopal.vercel.app](https://portfolio-dhanya-jayagopal.vercel.app)

---

## What's here

- **Work** — Case studies from Bomisco, Docusign, and LeanIX (SAP)
- **Writing** — Essays on product, AI in enterprise, and craft
- **About** — Background, trajectory, and what I'm currently thinking about
- **Now** — A single page updated as things change

## Stack

- **Framework:** Next.js 16, React 19, TypeScript, App Router
- **Design:** Atelier — a custom editorial-minimal CSS system (no Tailwind). Instrument Serif + Geist + Geist Mono
- **Content:** Markdown files in `content/` with gray-matter frontmatter
- **Deployment:** Vercel

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

Content lives in `content/work/` and `content/writing/` as Markdown files. The home page copy is in `content/home.md`.
